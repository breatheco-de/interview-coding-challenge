# `04` La palabra más larga

Haga que la función `LongestWord(sen)` tome el parámetro `sen` que se le pasa y devuelva la palabra más larga de la cadena. Si hay dos o más palabras con la misma longitud, devuelva la primera palabra de la cadena con esa longitud. Ignore la puntuación y asuma que `sen` no estará vacío. Las palabras también pueden contener números, por ejemplo "Hello world123 567".

## Ejemplos

```
Input: "fun&!! time"
Output: time
```

```
Input: "I love dogs"
Output: love
```
