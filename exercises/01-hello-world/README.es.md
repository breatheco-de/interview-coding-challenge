# `01` Primer Ejercicio

Hemos creado este primer ejercicio como ejemplo; lo puedes encontrar en la carpeta `./01-hello-world`.

1. Cada ejercicio debe estar ubicado en una carpeta separada y debe contener un archivo `README.md` con las instrucciones del ejercicio escritas en Markdown.
2. También puedes añadir un archivo `README.[lang].md` para traducciones, por ejemplo: `README.es.md` para español.
3. En la misma carpeta encontrarás un archivo `app.js` donde deberás escribir la solución.

## Cómo resolver el ejercicio

1. Escribe el código que resuelva el problema del ejercicio (Este primer ejercicio ya ha sido resuelto).
2. Cuando lo necesites, puedes comprobar si tu código se ejecuta usando el botón  `Ejecutar mi códgo` en la opción `Ejecutar` (fondo de la vista).
3. Una vez que hayas terminado y tu código se ejecute y resuelva el problema, pulsa el botón `Test` para verificar tu solución.
4. Si obtienes un resultado en verde, puedes continuar (¡y celebrar!).
