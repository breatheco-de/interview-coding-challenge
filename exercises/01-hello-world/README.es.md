# `01` Primer Ejercicio

Hemos creado este primer ejercicio como ejemplo. Lo puedes ubicar en la carpeta `./01-hello-world`.

1. Cada ejercicio estará ubicado en una carpeta propia 
2. Encontrarás un archivo `README.md` con las instrucciones del ejercicio escrito en _markdown_, el cual será como una página de un libro, sin archivos de código.
3. También encontrarás un archivo app.js donde deberás escribir la solución.
