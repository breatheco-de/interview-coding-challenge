# `05` Cambios de letra

Haga que la función `LetterChanges(str)` tome el parámetro `str` que se le pasa y lo modifique usando el siguiente algoritmo.

Reemplace cada letra en la cadena por la letra siguiente en el alfabeto (por ejemplo, `c` se convierte en `d`, `z` se convierte en `a`).

Luego capitalice cada vocal en esta nueva cadena (`a`, `e`, `i`, `o`, `u`) y finalmente devuelva esta cadena modificada.
